# Автоматизоване тестування алгоритму TripleDES

Цей проєкт присвячений автоматизованому тестуванню криптографічних бібліотек для Python, що реалізують алгоритм шифрування TripleDES (3DES), а також інтеграції цих тестів у процеси безперервної інтеграції та розгортання (CI/CD).

## Основні цілі
- Створення набору автоматизованих тестів для перевірки коректності роботи алгоритму TripleDES.
- Налаштування CI/CD конвеєра для автоматичного запуску тестів при змінах в коді.
- Забезпечення високої якості та безпеки криптографічних операцій.

## Технології
- **Мова програмування:** Python
- **Тестування:** `pytest` (або інший фреймворк)
- **CI/CD:** Підтримка інтеграції з популярними CI-системами.

## Встановлення та запуск

1. Створіть та активуйте віртуальне середовище:
   ```bash
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # Linux/MacOS:
   source venv/bin/activate
   ```

2. Встановіть необхідні залежності (після їх додавання):
   ```bash
   pip install -r requirements.txt
   ```

3. Запуск тестів:
   ```bash
   pytest
   ```
